# Femeter v0.1.0 Release Notes

**发布日期:** 2026-04-03

## 核心功能

✅ **完整 DLMS/COSEM 协议栈** — 12 个 crate，覆盖从 HDLC 帧到 COSEM 接口类的全部层级
✅ **三相电能计量** — 支持 ATT7022E 和 RN8302B 芯片
✅ **多通道通信** — RS485 (HDLC) + 红外 (IEC 62056-21) + LoRaWAN
✅ **电能质量监测** — 谐波分析、THD、功率因数
✅ **防窃电检测** — 4 种检测模式
✅ **OTA 双 Bank 升级** — 安全固件更新，掉电保护
✅ **费率/负荷管理** — TOU 分时计费、负荷曲线、最大需量
✅ **虚拟电表模拟器** — 桌面测试环境，无需硬件
✅ **FreeRTOS 集成** — 基于 FM33A068EV (Cortex-M0+)
✅ **生产工具** — 校表、出厂测试、J-Link 调试

## 已知限制

- 固件尚未在真实硬件上进行完整集成测试
- LoRaWAN Class C 模式需要实际网关验证
- 部分计量芯片驱动的校准参数需根据实际硬件调整
- virtual-meter 的 TCP 服务器为简化实现，非完整 DLMS 服务端
- 文档和注释仍有部分模块需要完善
