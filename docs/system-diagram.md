## 电源架构

```
┌─────────────────────────────────────────────────────────┐
│                       电源系统                           │
│                                                          │
│  220VAC ───┬─── 隔离反激 ──── 12VDC ──── LDO(3.3V) ────│─── VDD 3.3V
│            │                               ME6211       │    (MCU + 计量 + LCD)
│            │                                            │
│            │                 12VDC ──── LDO(1.8V) ────│─── 1.8V
│            │                               HT7131       │    (EC800N IO)
│            │                                            │
│            │             ┌──────────────┐               │
│            └─────────────│ ER26500 电池  │──────────────│─── VBAT 3.6V
│  掉电检测 ←──── SVD ────│ 3.6V 8500mAh │               │    (RTC + LCD + SRAM)
│                          │ Li-SOCl₂     │               │
│                          └──────┬───────┘               │
│                                 │                        │
│                          ┌──────▼───────┐               │
│                          │ 0.47F 法拉   │               │
│                          │ 电容 (缓冲)  │               │
│                          └──────────────┘               │
│                                                          │
│  掉电后:                                                 │
│    MCU → DEEPSLEEP (1.5µA)                              │
│    LCD → 保持显示 (~3µA)                                 │
│    RTC → 走时 (~0.5µA)                                  │
│    总计: ~5µA                                            │
│    续航: 8500mAh / 5µA ≈ 194 年 (理论)                  │
│    实际: ≥ 30 天 (LCD显示) / ≥ 5 年 (RTC)               │
└─────────────────────────────────────────────────────────┘
```

## Flash 分区

```
512KB Flash:
┌──────────┬──────────────────┬──────────────────┬──────────┐
│ Boot     │ Normal           │ OTA              │ Param    │
│ 16KB     │ 128KB            │ 128KB            │ 16KB     │
│ 0x00000  │ 0x04000          │ 0x24000          │ 0x44000  │
│          │                  │                  │          │
│ Bootldr  │ 应用固件          │ OTA备份          │ 校表参数  │
│ 版本校验  │ DLMS/COSEM      │ CRC32校验        │ 费率表   │
│ OTA搬运  │ MQTT/TLS        │ 搬运成功后清除    │ 序列号   │
│ 回滚     │ 计量/通信/显示    │                  │ 脉冲常数  │
│ LED指示  │ RTOS/Embassy    │                  │ OTA计数  │
└──────────┴──────────────────┴──────────────────┴──────────┘
剩余: 512K - 16K - 128K - 128K - 16K = 224K (保留扩展)

外部 W25Q64 (8MB, 可选):
┌──────────┬──────────┬──────────┬──────────┐
│ DLMS对象  │ 日志     │ OTA缓存  │ 预留     │
│ 1MB      │ 2MB      │ 2MB      │ 3MB      │
└──────────┴──────────┴──────────┴──────────┘
```
